# ecommerce
🛒 EcommerceLaravel

Aplikasi E-Commerce berbasis Laravel yang dirancang untuk mengelola proses penjualan produk secara online, mulai dari manajemen produk, kategori, keranjang belanja, hingga transaksi pembelian.

Project ini dibuat sebagai implementasi pembelajaran Web Development dengan Laravel Framework.

✨ Fitur Utama

🔐 Autentikasi Login & Register

🗂️ Manajemen Kategori Produk

📦 Manajemen Produk

🛍️ Keranjang Belanja (Cart)

💳 Proses Checkout

🧾 Riwayat Transaksi

👤 Role Admin & User

📊 Dashboard Admin

🗃️ Manajemen Database MySQL

🛠️ Teknologi yang Digunakan

Laravel

PHP

MySQL

Bootstrap
Composer
XAMPP
